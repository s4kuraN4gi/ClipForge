## 審査員1: EC事業コンサルタント（評価: C-）

**結論: 価格は競争力がある。だが「安いから選ばれる」のではなく「できることが少なすぎて選ばれない」リスクが高い。**

ClipForgeスターター ¥1,980 vs Pippit ¥2,834 vs Creatify ¥2,850。**価格だけ見れば最安クラス**。ここは心配不要。

問題は**得られる価値の差**:

**¥1,980で月15本。しかし8秒×3テンプレート = 実質3パターンしか作れない。** 15本生成できても、同じ3テンプレートの繰り返し。4本目以降を作る動機がない。

> 「月15本使えます」と言いながら、実際に使い切るユーザーはいない。これは**見かけ倒しの上限**。競合は本数が少なくても1本あたりの価値が高い。

## 審査員2: SaaS料金設計アナリスト（評価: C+）

**結論: 粗利率は健全。だが「本数」で売るモデル自体に問題がある。**

**粗利計算（WaveSpeed ¥45/本）:**

| プラン | 全本数消化時の粗利 | 半分消化時（実態に近い） |
|---|---|---|
| スターター 15本 | ¥1,980 - ¥675 = **¥1,305（65.9%）** | ¥1,980 - ¥338 = **¥1,642（82.9%）** |
| ビジネス 25本 | ¥4,980 - ¥1,125 = **¥3,855（77.4%）** | ¥4,980 - ¥563 = **¥4,417（88.7%）** |

粗利率は十分。**問題は料金モデルの設計思想。**

1. **「月○本」は心理的ハードル** — ハンドメイド作家は月に何十種類も商品を出さない。minne/Creema作家の平均出品数は10-30品。新商品は月1-3品。**月15本も必要ない。**
2. **本数制限は「使わなきゃ損」感を生まない** — Netflixやジムと違い、動画生成は能動的な作業。「今月あと12本使える」と言われても行動動機にならない。
3. **従量課金の方が合理的かもしれない** — ¥200-300/本の都度課金 or ¥500/月+¥150/本のハイブリッドの方が、低単価ユーザーの裾野を広げられる。

**料金モデル案:**

| モデル案 | 構成 | メリット |
|---|---|---|
| 現状維持 | ¥1,980/15本 | 分かりやすい |
| **都度課金** | ¥300/本（買い切り） | 心理的ハードル最低 |
| **ハイブリッド** | ¥980/月（5本付き）+追加¥200/本 | エントリー価格を下げつつ粗利確保 |
| **クレジット制** | ¥1,980で50クレジット（動画=5cr、テンプレート選択=無料） | Pippitと同じモデル。拡張性あり |

## 審査員3: 日本ハンドメイド市場スペシャリスト（評価: D+）

**結論: ターゲットの実態を見ていない。ハンドメイド作家は「動画が欲しい」のではなく「売上が欲しい」。**

**日本のハンドメイド作家の実態:**
- minne作家数: 約89万人（大半は月売上1万円以下）
- 主な販促手段: Instagram投稿、minne内SEO、口コミ
- 動画制作: **スマホで撮影→CanvaかCapCut（無料版）で編集→Instagram Reelsに投稿**
- 月の販促費用: **ほぼゼロ（多くても数千円）**

**致命的な問題:**
1. **¥1,980/月は「高い」** — 月売上5万円以下の作家にとって、売上の4%以上が動画ツールに消える。Canva Pro ¥691で十分と思っている層が大半。
2. **「AI動画」の価値が伝わらない** — 「写真1枚→8秒の動画になります」と言われて「それで売上が上がる」と確信できる作家がいない。ROIの実証がゼロ。
3. **8秒は短すぎる** — Instagram Reelsの推奨は15-90秒。TikTokも9秒以上推奨。**8秒の動画はSNSのアルゴリズムに不利。**
4. **テンプレート3種は致命的** — 作家は「自分の世界観」を大事にする。3種では「自分のブランドに合わない」で離脱。最低10-15種は必要。

**ハンドメイド作家が本当に欲しいもの:**
- 「この動画を投稿したら売上が上がった」という**成功事例**
- 自分の世界観に合った**おしゃれなテンプレート**
- **Instagram Reelsに最適な15-30秒の動画**
- **無料で試せる**十分な体験（1本では判断できない）

> 「月¥1,980を払ってでも使いたい」と思わせるには、ROIの実証が不可欠。『ClipForgeで作った動画を投稿したら、いいね数が3倍になった』『注文が20%増えた』——このエビデンスなしに有料転換は極めて困難。

## 審査員4: VC投資判断パートナー（評価: C）

**結論: ニッチは正しい。だが現状の機能と価格では「¥1,980を払う理由」が弱すぎる。**

**ポジティブ面（競合にない強み）:**
1. **ニッチが空いている** — 「日本のハンドメイド作家向けAI動画」は誰もやっていない。Pippitは英語圏EC、heroshはアパレル特化。**空白地帯は実在する。**
2. **価格は最安クラス** — ¥1,980は競合比で安い。価格で負けているわけではない。
3. **AIコスト¥45/本は健全** — WaveSpeed移行で粗利率65-77%。SaaSとして十分な水準。
4. **個人開発の機動力** — 大手が入りにくい「月数万円のニッチ市場」は個人開発者に最適。

**ネガティブ面（投資判断としてのリスク）:**

| リスク | 深刻度 | 理由 |
|---|---|---|
| **テンプレート3種** | 致命的 | 4本目以降の生成動機がない。月額モデルが成立しない |
| **動画8秒** | 致命的 | SNSのアルゴリズム推奨を下回る。ユーザーに実害 |
| **minne/GMOペパボの動向** | 高 | minne親会社がColorMe向けに動画LP自動生成を開始。プラットフォーム内製化のリスク |
| **Amazon動画生成（無料）** | 高 | 現在US限定だが日本展開で一部ユーザー流出 |
| **ROI実証ゼロ** | 高 | 「これを使えば売上が上がる」の証拠がない |

**利益シミュレーション（現実的シナリオ）:**

| 指標 | 楽観 | 現実的 | 悲観 |
|---|---|---|---|
| 月間サイト訪問 | 3,000 | 1,000 | 300 |

| 月売上 | ¥29,700 | ¥1,980 | ¥0 |
| インフラ固定費 | -¥5,500 | -¥5,500 | -¥5,500 |

## 全員一致の指摘

1. **テンプレート3種は全員が「致命的」と判定** — これが解決しない限り月額課金は成立しない
2. **動画8秒はSNS推奨以下** — 最低15秒、できれば30秒対応が必要
3. **価格自体は問題ない** — ¥1,980は競合比で安い。問題は「¥1,980の価値がない」こと
4. **ROI実証を最優先で作るべき** — 実際のハンドメイド作家に使ってもらい、売上変化のデータを取る

## 日本のハンドメイド作家が実際に使っているツール

1. **Canva** — デザイン全般（画像・バナー・簡易動画）。Pro契約者多数
2. **CapCut（無料版）** — TikTok/Reels編集。Commerce Proではなく無料アプリ
3. **Promeo / InShot** — モバイル動画編集。テンプレートドリブン
4. **Vrew** — 字幕自動生成。YouTube制作で人気
5. **スマホ標準ツール** — iPhone内蔵エディタ、Instagram編集機能

**重要な洞察**: ハンドメイド作家はAI動画生成ツールをまだ使っていない。「写真1枚からAI動画」というコンセプト自体がこの市場では新しい。市場教育が必要。
